## 프로젝트 소개

<악마와의 계약은 혼인신고로!>는 악마 신화에 심취한 주인공이 왕국의 금서관에서 몰래 대악마를 소환한 뒤, 그와 결혼하기 위해 마법진을 완성하고 다양한 악마들을 수집·사육하는 병맛 오컬트 퍼즐 게임입니다.

플레이어는 한붓그리기 퍼즐처럼 마법진을 그려 악마를 소환해야 하며, 제한된 자원과 정신력을 관리하면서 각 스테이지를 클리어해야 합니다.

## 주요 기능

### 🔯 마법진 시스템

플레이어는 마법진을 직접 그려 악마를 소환합니다.

마법진을 그리는 과정에서 잘못된 획을 긋거나, 실행 취소를 사용하거나, 의도치 않은 개체를 소환할 경우 정신력이 감소합니다. 정신력이 모두 소진되면 게임 오버가 됩니다.

또한 마법진을 그릴 때 필요한 자원인 **쥐의 피**는 한정되어 있기 때문에, 플레이어는 최소한의 획으로 마법진을 완성해야 합니다. 자원이 부족한 경우 실행 취소를 통해 이전 단계로 되돌아가 다시 시도할 수 있지만, 이 과정에서도 정신력이 소모됩니다.

### 📖 악마 도감 시스템

플레이어는 다양한 악마를 수집할 수 있습니다.

각 악마는 고유한 능력을 가지고 있으며, 특정 스테이지에서 플레이어를 도와주는 역할을 합니다. 수집한 악마들은 도감에서 확인할 수 있습니다.

### 💍 대악마 청혼 조건 시스템

주인공은 대악마와 결혼하기 위해 여러 청혼 조건을 충족해야 합니다.

플레이어는 퍼즐을 해결하고 악마를 수집하며, 대악마가 요구하는 조건을 하나씩 맞춰나가야 합니다.
